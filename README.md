# Excel Matcher (엑셀 거래처/고객 매칭 프로그램)

## 프로그램 목적

엑셀 A의 거래처/고객 데이터와 엑셀 B의 데이터를 비교하여, **같은 거래처인지 유사도 기반으로 판단**하고 **고객명 일치 여부**를 별도로 확인한 뒤 결과 엑셀 C를 생성합니다.

일반 사용자는 Python 설치 없이 `excel_matcher.exe` 하나만 실행하면 됩니다.

---

## 폴더 구조

```text
excel_matcher/
 ├─ excel_matcher.exe      # 배포용 실행파일 (build.bat 실행 후 생성)
 ├─ main.py                # 프로그램 진입점
 ├─ config.py              # 경로/기본 설정
 ├─ excel_io.py            # 엑셀 입출력
 ├─ normalizers.py         # 정규화 규칙
 ├─ scorers.py             # 점수 계산
 ├─ matcher.py             # 매칭 로직
 ├─ requirements.txt
 ├─ build.bat              # exe 빌드
 ├─ run.bat                # exe 실행
 ├─ input/
 │   ├─ A.xlsx
 │   └─ B.xlsx
 ├─ output/
 │   └─ C.xlsx             # 실행 후 생성
 ├─ logs/
 │   └─ run_YYYYMMDD_HHMMSS.log
 ├─ rules/
 │   ├─ sido_alias.json
 │   └─ company_remove_words.json
 ├─ README.md
 └─ 실행메뉴얼.md
```

---

## 입력 / 출력

### 입력

- `input/A.xlsx`
- `input/B.xlsx`

필수 컬럼(동일, **아래 순서 권장**):

| 순서 | 컬럼명 | 유사도 사용 | 설명 |
|------|--------|-------------|------|
| 1 | 거래처코드 | X | 거래처 고유 키 (C 파일에 A/B 원본값 전달) |
| 2 | 거래처명 | O | 거래처 유사도 계산 |
| 3 | 시도명 | O | 거래처 유사도 계산 |
| 4 | 시군구명 | O | 거래처 유사도 계산 |
| 5 | 도로명주소 | O | 거래처 유사도 계산 |
| 6 | 전화번호 | O | 거래처 유사도 계산 |
| 7 | 고객코드 | X | 고객 고유 키 (C 파일에 A/B 원본값 전달) |
| 8 | 고객명 | O | 100% 일치 여부 판정 (가점 조건) |
| 9 | 진료과목명 | X | 동명이인 구분 참고용 (C 파일에 A/B 원본값 전달) |

> `거래처코드`, `고객코드`, `진료과목명`은 유사도 점수 계산에 사용하지 않습니다.

### 출력

- `output/C.xlsx`

주요 결과 컬럼:

- A/B 원본 컬럼 9개 (접두사 `(A)`, `(B)`) — 거래처코드, 거래처명, 시도명, 시군구명, 도로명주소, 전화번호, 고객코드, 고객명, 진료과목명
- 거래처 기본 유사도점수
- 고객명 가점
- 거래처 최종 유사도점수
- 거래처 일치여부 (`매칭` / `검수필요` / `미매칭`)
- 고객명 일치여부 (`일치` / `불일치`)
- 근접후보 1~3 (대표 후보와 5점 이내, 최대 3건)

---

## 점수 계산 방식

### 거래처 기본 유사도점수 (100점 만점, 고객명 제외)

| 항목 | 가중치 |
|------|--------|
| 거래처명 | 45% |
| 시도명 | 15% |
| 시군구명 | 15% |
| 전화번호 | 20% |
| 도로명주소 | 5% |

```text
base_score = (
    company_score * 0.45 +
    sido_score * 0.15 +
    sigungu_score * 0.15 +
    phone_score * 0.20 +
    address_score * 0.05
)
```

### 고객명 가점

- 조건: `base_score >= 75` **그리고** 고객명 100% 일치
- 가점: **+3점**
- 최종 점수: `min(100, base_score + bonus)`

> 고객명은 거래처 기본 유사도 계산에 포함하지 않습니다.
> `진료과목명`은 동명이인 구분을 위한 참고 정보이며, 유사도 계산 및 일치 판정에 사용하지 않습니다.

### 매칭 기준

**거래처 일치여부**

| 점수 | 판정 |
|------|------|
| 90점 이상 | 매칭 |
| 80~89점 | 검수필요 |
| 80점 미만 | 미매칭 |

**고객명 일치여부**

- 정규화 후 100% 일치 → `일치`
- 그 외 → `불일치`

---

## 정규화 규칙 요약

| 항목 | 처리 |
|------|------|
| 거래처명 | 공백/특수문자 제거, 불용어 제거 (`rules/company_remove_words.json`) |
| 시도명 | 별칭 매핑 (`rules/sido_alias.json`) |
| 시군구명 | 공백 제거, 끝 `시/군/구` 제거 |
| 전화번호 | 숫자만 추출 후 규칙 점수화 |
| 도로명주소 | 공백/특수문자 제거 |
| 고객명 | 공백/특수문자 제거, 100% 일치만 인정 |

### 전화번호 점수

- 완전 동일: 100점
- 시도명 같고 한쪽 번호가 다른 쪽 끝과 일치: 100점
- 끝 4자리 동일: 70점
- 그 외: 0점

---

## 후보 선택 규칙

1. A 각 행마다 B 전체와 비교
2. **시도명이 같은 B 행**을 우선 후보로 사용 (없으면 B 전체 사용)
3. 최고 `거래처 최종 유사도점수` 후보를 대표 매핑으로 선택
4. 동점 시 우선순위: 고객명 일치 → 전화번호 점수 → 거래처명 점수 → 주소 점수
5. 대표 후보와 **5점 이내** 후보를 근접후보로 최대 3개 출력

## 규칙 추가 방법

### 시도명 별칭 추가

`rules/sido_alias.json` 편집:

```json
{
  "서울특별시": "서울",
  "경기도": "경기"
}
```

### 거래처명 불용어 추가

`rules/company_remove_words.json` 편집:

```json
[
  "학교법인",
  "주식회사",
  "(주)"
]
```

JSON 형식 오류가 있으면 프로그램이 실행을 중단하고 오류 메시지를 표시합니다.

## 개발 환경

- Python 3.10+
- Windows 10/11

### 의존성 설치

```bat
pip install -r requirements.txt
```

### 개발 모드 실행

```bat
python main.py
```

---

## exe 빌드 방법

프로젝트 폴더에서:

```bat
build.bat
```

또는 직접:

```bat
pyinstaller --onefile --name excel_matcher main.py
copy dist\excel_matcher.exe excel_matcher.exe
```

빌드 전 `pip install pyinstaller` 가 필요합니다.

---

## 배포 방법

아래 파일/폴더를 **같은 폴더**에 묶어 배포합니다.

```text
excel_matcher.exe
input/
output/
logs/
rules/
  ├─ sido_alias.json
  └─ company_remove_words.json
실행메뉴얼.md
```

배포 시 `input/A.xlsx`, `input/B.xlsx`는 사용자가 직접 준비합니다.

> `rules` 폴더는 exe와 **같은 위치**에 있어야 합니다. exe 내부가 아니라 외부 폴더로 동작합니다.

---

## 로그

실행 시 `logs/run_YYYYMMDD_HHMMSS.log` 파일이 생성되며, 콘솔과 동일한 내용이 기록됩니다.

- 실행 시작/완료 시간
- 입력 파일 경로
- 데이터 건수
- 진행률
- 오류 메시지

---

## 자주 발생하는 오류

| 오류 | 해결 |
|------|------|
| A.xlsx / B.xlsx 없음 | `input` 폴더에 파일명 정확히 배치 |
| 필수 컬럼 누락 | 9개 컬럼명 확인 |
| rules JSON 없음/오류 | `rules` 폴더 및 JSON 형식 확인 |
| C.xlsx 저장 실패 | 엑셀에서 C.xlsx 닫고 재실행 |

자세한 사용법은 `실행메뉴얼.md`를 참고하세요.
